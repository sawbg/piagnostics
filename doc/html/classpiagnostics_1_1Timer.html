Detailed Description

This class is designed to act as a timer for a series of commands in a while loop.

Note that this timer can be up to one second too short or too long.

In Piagnostics, this class times the various groups' diagnostic information display time.

And example use of Timer: 

Timer timer(5);  // initialize Timer instance, 5 sec time

while(timer) {
DoStuff();
DoMoreStuff();
Chillax();
}
